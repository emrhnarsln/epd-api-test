Privacy Policy

Last updated: May 15, 2026

Overview

The EPD Course Assistant helps students access course content, summaries, and quiz questions through an external API.

Data Sent to the API

When a user asks about a lesson, quiz, course title, language, or topic, the assistant may send limited query information to the API, such as: lesson number, language code, search terms, requested quiz count, and whether answers should be shown.

Personal Data

We do not intentionally collect personal data through this API. Users should not enter sensitive personal information into the assistant.

How Data Is Used

Query information is used only to retrieve relevant educational content and quiz data from the course dataset.

Data Storage

The API does not intentionally store user queries or personal information. Hosting providers may process technical logs for security, diagnostics, and service operation.

Third-Party Services

This assistant uses an API hosted on Google Cloud Run. Use of ChatGPT is subject to OpenAI's applicable terms and privacy practices.

Contact

For privacy questions, contact the owner or administrator of this GPT.
